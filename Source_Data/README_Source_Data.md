# Source Data for "Decoding cell signaling via optimal transport and information theory"

This folder contains the numerical input data and processed output data underlying the main and supplementary figures. It is organized for Nature Communications source-data review. The index file `Source_Data_Index.xlsx` provides the figure-level map and file manifest in Excel format.

## Folder structure

- `raw_input_data/`: experimental numerical inputs used by the analysis scripts.
- `processed_output_data/`: numerical outputs generated by the analysis scripts and used to plot main and supplementary figures.
- `Source_Data_Index.xlsx`: Excel workbook containing the figure-level source-data map and complete file manifest.

## Raw input data descriptions

### TNF data

`raw_input_data/TNF/30min/` and `raw_input_data/TNF/4hr/` contain tab-delimited input files. Columns are:

1. TNF dose in ng/mL.
2. Conditional mean response, mean Z given X, in arbitrary units.
3. Conditional standard deviation of response, standard deviation Z given X, in arbitrary units.

`raw_input_data/TNF/target_MI/` contains MI values used as target values for the TNF input-output reconstruction.

### RAS-MAPK SOS-RAF data

`raw_input_data/RAS_MAPK_SOS_RAF/` contains single-cell time-series CSV files. In each file, the first column is time after EGF stimulation and the remaining columns are single-cell trajectories. SOS is used as input X and RAF is used as output Z. `DMSO` indicates control; `10nMMEKi` indicates MEK-inhibited condition.

## Processed output data descriptions

### Theoretical data

`processed_output_data/Theoretical/` contains motif-level numerical outputs for informational fidelity, geometric fidelity, MI, 2-Wasserstein distance, optimized noise parameters, and related quantities. Files containing `lambda-W-I-rho` include lambda, 2-WD, MI, and correlation. Files containing `geofid-infofid` or `W-I` contain geometric fidelity and informational fidelity values used for fidelity plots. The `C1FFL/h-2/` folder contains the cooperative-binding C1-FFL check with Hill coefficient h=2.

### TNF processed data

`processed_output_data/TNF_30min/` and `processed_output_data/TNF_4hr/` contain:

- `MI_WD_summary/`: MI and 2-WD values with uncertainty.
- `conditional_marginal_joint_distributions/`: conditional distributions, input marginals, output marginals, and joint distributions.
- `cdf_quantile_functions/`: CDF and quantile-function data used for 2-WD calculation and plotting.
- `convergence/`: numerical convergence diagnostics.
- `mapping_factors_and_fit_curves/`: Hill/linear fit curves and mapping-factor estimates.

### RAS-MAPK processed data

`processed_output_data/RAS_MAPK_SOS_RAF/` contains the processed numerical outputs used for the SOS-RAF/RAS-MAPK analysis, including time series, marginal distributions, joint distributions, dynamic range, correlation, mean, standard deviation, MI, 2-WD, and inverse 2-WD. Marginal-distribution files begin with `Marginal_dist_`; joint-distribution files begin with `Joint_dist_`.

### Non-Gaussian illustrative example

The numerical data for the illustrative unimodal-to-bimodal relay example are included within `processed_output_data/Theoretical/Gauss-nonGauss-Compare/`.

## Notes

- DMSO in the file names stand for the `control condition` and Drug for the `MEKi cobdition`.
